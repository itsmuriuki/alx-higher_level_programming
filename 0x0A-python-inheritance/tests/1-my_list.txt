=======================
How to use 1-my_list.py
=======================

This function takes a list and prints the list sorted.
Function assumptions: all entries are integers

     >>> MyList = __import__('1-my_list').MyList

=============

     >>> module_doc = __import__('1-my_list').__doc__
     >>> print(len(module_doc) > 0)

     >>> print(len(class_doc) > 0)
     True

     >>> function_doc = __import__('1-my_list').MyList.print_sorted.__doc__
     >>> print(len(function_doc) > 0)
     True


Positive Integers
=================

     >>> my_list = MyList()
     >>> my_list.append(1)
     >>> my_list.append(4)
     >>> my_list.append(2)
     >>> my_list.append(3)
     >>> my_list.append(5)
     >>> print(my_list)
     [1, 4, 2, 3, 5]
     >>> my_list.print_sorted()
     [1, 2, 3, 4, 5]
     >>> print(my_list)
     [1, 4, 2, 3, 5]

     >>> my_list = MyList()
     >>> my_list.append(1000)
     >>> my_list.append(4000)
     >>> my_list.append(2000)
     >>> my_list.append(3000)
     >>> my_list.append(5000)
     >>> print(my_list)
     [1000, 4000, 2000, 3000, 5000]
     >>> my_list.print_sorted()
     [1000, 2000, 3000, 4000, 5000]
     >>> print(my_list)
     [1000, 4000, 2000, 3000, 5000]
