How to use 3-say_my_name.py
===========================

	The function ``say_my_name()`` will display "My name is" followed by the first
	and optional last name supplied to it

Strings
=======
	The first and last name inputs must be strings

	>>> say_my_name("Gerald", "Muriuki")
	My name is Gerald Muriuki

	>>> say_my_name("John", "Mwangi")
	My name is John Smith

	>>> say_my_name("Walter", "mzii")
	My name is Walter White

	>>> say_my_name("Bob")
    	My name is Bob

	>>> say_my_name("Gerald")
    	My name is Gerald

	>>> say_my_name("D", "K")
	My name is D K

	>>> say_my_name("    ", "    ")
	My name is

	>>> say_my_name(" ", " ")
	My name is

	>>> say_my_name(" ", Muriuki"")
	My name is   Muriuki

	>>> say_my_name("Gerald", " ")
   	My name is Gerald

	>>> say_my_name("", "")
	My name is

	>>> say_my_name("Derek", "")
	My name is Gerald

	>>> say_my_name("", "Muriuki")
	My name is  Muriuki

	Characters

	>>> say_my_name('G', 'M')
	My name is G M

   	>>> say_my_name("Gerald", 'M')

   	>>> say_my_name('G', "Muriuki")
